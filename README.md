# Fairy Tales App

A Next.js application for browsing fairy tales stored in a local SQLite database.

## Features

- Hero page with links to random fairy tale and list of all fairy tales
- Random fairy tale page
- List of all fairy tales with links to individual pages
- Individual fairy tale pages by ID
- SQLite database for storing fairy tales

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Initialize the database:
   ```bash
   npm run init-db
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To build for production:
```bash
npm run build
npm start
```
